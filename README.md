# Algorithm & Data Structure Lab

This repository contains lab assignments for algorithm and data structure practice using C. Each lab folder includes one or more task implementations, organized by date and lab number.

## Repository Structure

- `25_09_25-Task1/` - Lab 1 implementation for student detail input.
- `25_10_02-Task1-2-3/` - Lab 3 implementations with two tasks: matrix sorting and median finding.
- `25_10_09-Task1/` - Lab 2 implementation for node and pointer practice.
- `25_10_16-Task1/` - Lab 3 implementation for node and pointer with user input.

## Notes

- All tasks are implemented in C source files (`*.c`).
- Compile with `gcc` and run each task from its folder.
- Some task folders may contain multiple implementations or variations for practice.

## How to Use

1. Open a terminal in the repository root.
2. Change to the desired task folder, for example:
   ```bash
   cd 25_09_25-Task1
   ```
3. Compile the C file:
   ```bash
   gcc task_1.c -o task_1
   ```
4. Run the binary:
   ```bash
   ./task_1
   ```
